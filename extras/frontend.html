TinyPysics Server Test Interface

This is a simple test interface for the TinyPysics multiplayer server.

Result:

-

Available Commands:

  • newship x y id - Create new ship at position
  • getall - Get all object positions
  • newclient id - Register new client
  • delclient id - Remove client
  • boost id - Apply thrust
  • noboost id - Stop thrust
  • rotateleft id - Rotate left
  • rotateright id - Rotate right
